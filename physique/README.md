# Version physique

Cette section regroupe les documents destinés au jeu sur table. La version à mettre en avant est l’initiation quadratique.

## À utiliser en premier

Dossier actif : [`initiation/`](initiation/)

- [`initiation/regles/REGLES_INITIATION_QUADRATIQUE.md`](initiation/regles/REGLES_INITIATION_QUADRATIQUE.md) : règles complètes de la version d’initiation.
- [`initiation/cartes/CARD_SET_INITIATION_QUADRATIQUE.md`](initiation/cartes/CARD_SET_INITIATION_QUADRATIQUE.md) : composition des decks d’initiation.
- [`initiation/cartes/stack_ou_pas_stack_cartes_png_quadratique.zip`](initiation/cartes/stack_ou_pas_stack_cartes_png_quadratique.zip) : export PNG des cartes.
- [`initiation/impression/`](initiation/impression/) : PDF imprimables et plateau compact.

## Dossiers

- `initiation/` : version physique active, prête à jouer et à imprimer.
- `historique/` : anciennes versions et documents de travail conservés pour référence.

## État actuel

La version physique active du projet est **Stack ou pas Stack — initiation quadratique** :

- mémoire 11 / 11 ;
- victoire à 11 points ;
- bonus `B(R)=R²` ;
- casse au 7e cadre ;
- Stack Spike sur 4 ou 5 cadres ;
- decks symétriques de 26 cartes.

Les fichiers hérités de Neon Stack V5 sont rangés dans [`historique/neon-stack-v5/`](historique/neon-stack-v5/) pour ne pas mélanger la version jouable actuelle avec les archives.
