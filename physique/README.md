# Version physique

Cette section regroupe les documents destinés au jeu sur table.

## Dossiers

- `regles/` : règles d’initiation quadratique actuelles.
- `cartes/` : composition des decks d’initiation.
- `impression/` : fichiers imprimables utiles, dont le plateau compact disponible.
- `archives-v5/` : documents Neon Stack V5.4 et anciens fichiers de travail conservés pour référence.

## État actuel

La version physique active du projet est **Stack ou pas Stack — initiation quadratique** :

- mémoire 11 / 11 ;
- victoire à 11 points ;
- bonus `B(R)=R²` ;
- casse au 7e cadre ;
- Stack Spike sur 4 ou 5 cadres ;
- decks symétriques de 26 cartes.

Les planches définitives spécifiques à cette variante peuvent être ajoutées dans `impression/` lorsqu’elles sont régénérées.
