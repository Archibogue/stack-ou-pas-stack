# Version internet

Cette section contient la version jouable en navigateur.

## Lancement local

Ouvrir simplement :

```text
internet/index.html
```

## Version V2

La nouvelle version web V2 est disponible dans :

```text
internet/v2/public/index.html
```

Elle propose une architecture modulaire, un moteur centralisé, un journal d’actions, une aide intégrée et une API PHP/MySQL optionnelle.

Le fichier fonctionne sans serveur et sans installation.

## Fonctionnalités V1.1

- deux joueurs sur le même écran ;
- decks d’initiation quadratique ;
- ordre obligatoire du tour : mise à jour → pioche → conception → fin de tour ;
- mémoire libre, mémoire totale, mémoire utilisée ;
- fonctions actives, piles de cadres, cadres parasites ;
- sauvegarde locale dans le navigateur ;
- export/import JSON d’une partie.
